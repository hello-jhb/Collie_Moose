# Source Authority Guidance

Prefer sources in this rough order, adjusted for workbook context:
- summary / one pager for final presented deal metrics
- sources and uses or debt schedule for capital structure
- assumptions / inputs for model assumptions
- cash flow / model sheets for row-derived outputs
- debt tabs for loan terms
- operating tabs for NOI support
- sensitivity tables only when the question asks for scenario or sensitivity

A source is more authoritative when it has clear labels, period headers, scenario/base case context, and formula lineage that matches the metric being claimed.

When source authority is ambiguous, return the uncertainty and alternatives rather than over-selecting a single fact.
