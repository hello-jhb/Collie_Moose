# NOI Guidance

NOI may appear as:
- Year 1 NOI
- stabilized NOI
- T12 NOI
- in-place NOI
- projected NOI
- exit year NOI
- annualized monthly NOI

Do not simply grab the first row labeled NOI.

For each NOI claim, provide:
- NOI type
- period
- source sheet/cell
- nearby row/column context
- reason this is the correct NOI for the current model question
- alternative NOI candidates, if any

Classify NOI candidates:
- historical actual NOI
- current run-rate NOI
- projected Year 1 NOI
- stabilized NOI
- exit year NOI
- annualized monthly NOI

For acquisition underwriting, important NOI values often include Year 1 projected NOI for going-in yield, stabilized NOI for stabilized yield, and exit year NOI for terminal value.

For refinance, important NOI values often include current, T12, or lender-underwritten NOI.

Always report the NOI type. If uncertain, return multiple NOI candidates rather than forcing one value.
