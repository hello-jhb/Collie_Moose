# Exit Assumptions Guidance

Exit assumptions may include exit cap rate, sale value, exit NOI, terminal value, disposition costs, reversion value, sale price, and residual value.

Do not select sale value by grabbing the first "Sale Price" label. Examine nearby rows and columns to determine whether the value is per unit, per key, gross sale price, net sale proceeds, unlevered reversion, levered proceeds, or a scenario/sensitivity value.

For terminal value, the common relationship is exit value equals exit year NOI divided by exit cap rate, adjusted for workbook-specific conventions. Report uncertainty when the workbook does not make this clear.

For each exit claim, include why the selected value is the correct exit assumption and list plausible alternatives.
