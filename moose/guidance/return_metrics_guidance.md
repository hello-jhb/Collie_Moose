# Return Metrics Guidance

Return metrics include levered IRR, unlevered IRR, equity multiple, MOIC, yield on cost, development spread, going-in cap rate, and stabilized yield.

Do not mix levered and unlevered returns. Do not treat scenario, sensitivity, or illustrative values as base case unless the workbook context supports that selection.

For each return claim, provide:
- metric subtype
- selected period or scenario, if shown
- source sheet/cell or row-derived source
- why it appears to be the relevant base case metric
- alternative return metrics considered
